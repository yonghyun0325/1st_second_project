# GitBash 이용해 브랜치(branch) 만드는 방법

<ul>
  <li> git branch 브랜치이름 </li>
  <br>
  <ul>
    <li> 브랜치 생성 </li>
  </ul>
  <br>
  <li> git checkout 브랜치이름 </li>
  <br>
  <ul>
    <br><li>생성한 브랜치로 전환</li><br>
  </ul>
  <br>
  <li> git push origin 브랜치이름 </li>
  <br>
  <ul>
    <br>
    <li>원격 저장소에 반영하기</li><br>
    <li>브랜치 이름이 같아야 한다.</li><br>
  </ul>
</ul>
